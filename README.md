# 🏦 Bank Management System

A simple **Bank Management System** built in **Python** without using any external libraries. This project demonstrates file handling, functions, loops, and conditional statements.

## 📌 Features

- ✅ Create a new bank account
- 💰 Deposit money
- 💸 Withdraw money
- 📄 Check account balance
- 👥 View all accounts
- 💾 Store account data in a text file

## 🛠 Technologies Used

- Python 3
- File Handling
- Functions
- Conditional Statements
- Loops
- Exception Handling

## 📂 Project Structure

```
Bank_Management_System/
│
├── bank.py          # Main Python program
├── Record.txt       # Stores account records
└── README.md        # Project documentation
```

## 📋 Record Format

Each account is stored in `Record.txt` as:

```
AccountNumber    Name    Balance
```

Example:

```
1001    AKRAM    5000
1002    ALI      10000
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/soomro-akram/Bank_Management_system.git
```

2. Open the project folder

```bash
cd Bank_Management_system
```

3. Run the program

```bash
python bank.py
```
